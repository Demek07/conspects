## **–°—Ä–∞–≤–Ω–µ–Ω–∏–µ `__init__` —Å –¥—Ä—É–≥–∏–º–∏ —Å–ø–µ—Ü–∏–∞–ª—å–Ω—ã–º–∏ –º–µ—Ç–æ–¥–∞–º–∏ (`__new__`, `__del__`)** ü§î

–í Python —Å—É—â–µ—Å—Ç–≤—É—é—Ç —Ç–∞–∫–∂–µ `__new__` –∏ `__del__`. `__new__` –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è —Å–æ–∑–¥–∞–Ω–∏—è –∏ –≤–æ–∑–≤—Ä–∞—Ç–∞ –Ω–æ–≤–æ–≥–æ –æ–±—ä–µ–∫—Ç–∞, –ø–µ—Ä–µ–¥ –≤—ã–∑–æ–≤–æ–º `__init__`. –í –Ω–µ–∫–æ—Ç–æ—Ä—ã—Ö —Å–ª—É—á–∞—è—Ö –º–æ–∂–µ—Ç –±—ã—Ç—å –ø–æ–ª–µ–∑–Ω–æ –ø–µ—Ä–µ–æ–ø—Ä–µ–¥–µ–ª–∏—Ç—å –º–µ—Ç–æ–¥ `__new__`, –¥–ª—è –∏–∑–º–µ–Ω–µ–Ω–∏—è –ø–æ–≤–µ–¥–µ–Ω–∏—è –æ–±—ä–µ–∫—Ç–æ–≤ –Ω–∞ —ç—Ç–∞–ø–µ —Å–æ–∑–¥–∞–Ω–∏—è –∏–ª–∏ –¥–ª—è –∏–∑–º–µ–Ω–µ–Ω–∏—è —Ç–∏–ø–∞ —Å–æ–∑–¥–∞–≤–∞–µ–º–æ–≥–æ –æ–±—ä–µ–∫—Ç–∞.

## **–õ—É—á—à–∏–µ –ø—Ä–∞–∫—Ç–∏–∫–∏ –∏ –æ–±—â–∏–µ –æ—à–∏–±–∫–∏ –ø—Ä–∏ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–∏ `__init__`** üåü

### –õ—É—á—à–∏–µ –ø—Ä–∞–∫—Ç–∏–∫–∏:

# 3. –ü—Ä–∏–Ω—Ü–∏–ø—ã —Å—Ä–∞–≤–Ω–µ–Ω–∏—è —Å—Ç—Ä–æ–∫ –≤ Python üêç

## –õ–µ–∫—Å–∏–∫–æ–≥—Ä–∞—Ñ–∏—á–µ—Å–∫–æ–µ —Å—Ä–∞–≤–Ω–µ–Ω–∏–µ —Å—Ç—Ä–æ–∫ –≤ Python üìö

–í Python, —Å—Ç—Ä–æ–∫–∏ —Å—Ä–∞–≤–Ω–∏–≤–∞—é—Ç—Å—è –ª–µ–∫—Å–∏–∫–æ–≥—Ä–∞—Ñ–∏—á–µ—Å–∫–∏. –≠—Ç–æ –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ —Å—Ç—Ä–æ–∫–∏ —Å—Ä–∞–≤–Ω–∏–≤–∞—é—Ç—Å—è, –∫–∞–∫ –µ—Å–ª–∏ –±—ã –æ–Ω–∏ –±—ã–ª–∏ —Å–ª–æ–≤–∞–º–∏ –≤ —Å–ª–æ–≤–∞—Ä–µ. –í –ø—Ä–æ—Ü–µ—Å—Å–µ —Å—Ä–∞–≤–Ω–µ–Ω–∏—è —ç–ª–µ–º–µ–Ω—Ç—ã –æ–±–µ–∏—Ö —Å—Ç—Ä–æ–∫ (—Ç.–µ. —Å–∏–º–≤–æ–ª—ã) —Å—Ä–∞–≤–Ω–∏–≤–∞—é—Ç—Å—è –¥—Ä—É–≥ —Å –¥—Ä—É–≥–æ–º "–ø–æ–ø–∞—Ä–Ω–æ". ‚úåÔ∏è

–°—Ä–∞–≤–Ω–µ–Ω–∏–µ –Ω–∞—á–∏–Ω–∞–µ—Ç—Å—è —Å –ø–µ—Ä–≤—ã—Ö —Å–∏–º–≤–æ–ª–æ–≤, –∏ –ø—Ä–æ–¥–æ–ª–∂–∞–µ—Ç—Å—è —Å –ø–æ—Å–ª–µ–¥—É—é—â–∏–º–∏, –ø–æ–∫–∞ –Ω–µ –±—É–¥–µ—Ç –æ–±–Ω–∞—Ä—É–∂–µ–Ω–∞ —Ä–∞–∑–Ω–∏—Ü–∞ –∏–ª–∏ –æ–¥–Ω–∞ –∏–∑ —Å—Ç—Ä–æ–∫ –∑–∞–∫–æ–Ω—á–∏—Ç—Å—è. –í —Å–ª—É—á–∞–µ –æ–±–Ω–∞—Ä—É–∂–µ–Ω–∏—è —Ä–∞–∑–Ω–∏—Ü—ã, —Å—Ç—Ä–æ–∫–∞ —Å –º–µ–Ω—å—à–∏–º —Å–∏–º–≤–æ–ª–æ–º —Å—á–∏—Ç–∞–µ—Ç—Å—è –º–µ–Ω—å—à–µ–π, –∞ —Å –±–æ–ª—å—à–∏–º —Å–∏–º–≤–æ–ª–æ–º ‚Äì –±–æ–ª—å—à–µ–π. –ï—Å–ª–∏ –æ–¥–Ω–∞ –∏–∑ —Å—Ç—Ä–æ–∫ –∑–∞–∫–æ–Ω—á–∏—Ç—Å—è, —Ç–æ –∫–æ—Ä–æ—Ç–∫–∞—è —Å—Ç—Ä–æ–∫–∞ —Å—á–∏—Ç–∞–µ—Ç—Å—è –º–µ–Ω—å—à–µ–π. üìè

## –ü—Ä–∏–º–µ—Ä—ã —Å—Ä–∞–≤–Ω–µ–Ω–∏—è —Å—Ç—Ä–æ–∫ —Å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ–º –ø–æ—Ä—è–¥–∫–æ–≤—ã—Ö –Ω–æ–º–µ—Ä–æ–≤ Unicode üî¢

```python
# –ü—Ä–∏–º–µ—Ä —Å—Ä–∞–≤–Ω–µ–Ω–∏—è –¥–≤—É—Ö —Å—Ç—Ä–æ–∫ —Å —Ä–∞–∑–Ω—ã–º–∏ —Å–∏–º–≤–æ–ª–∞–º–∏
```

## –°—Ä–∞–≤–Ω–µ–Ω–∏–µ –º–µ—Ç–æ–¥–æ–≤ `==` –∏ `is` –ø—Ä–∏ —Å—Ä–∞–≤–Ω–µ–Ω–∏–∏ —Å—Ç—Ä–æ–∫ üÜö

### 2.1 `__eq__` ‚Äì —Ä–∞–≤–µ–Ω—Å—Ç–≤–æ (==) ‚úÖ

`__eq__` –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç –ø–æ–≤–µ–¥–µ–Ω–∏–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä–∞ —Å—Ä–∞–≤–Ω–µ–Ω–∏—è "—Ä–∞–≤–Ω–æ" –¥–ª—è –≤–∞—à–µ–≥–æ –∫–ª–∞—Å—Å–∞. –≠—Ç–æ—Ç –º–µ—Ç–æ–¥ –¥–æ–ª–∂–µ–Ω –≤–æ–∑–≤—Ä–∞—â–∞—Ç—å `True`, –µ—Å–ª–∏ –æ–±—ä–µ–∫—Ç—ã —Ä–∞–≤–Ω—ã –∏–ª–∏ —ç–∫–≤–∏–≤–∞–ª–µ–Ω—Ç–Ω—ã, –∏ `False`‚Äù –≤ –ø—Ä–æ—Ç–∏–≤–Ω–æ–º —Å–ª—É—á–∞–µ.

–ü—Ä–∏–º–µ—Ä:

```python

    def __init__(self, radius):
        self.radius = radius
    
    def __eq__(self, other):
        if isinstance(other, Circle):
            return self.radius == other.radius
        return False

circle2 = Circle(10)
circle3 = Circle(20)

print(circle1 == circle3)  # False
```

### 2.2 `__ne__` ‚Äì –Ω–µ—Ä–∞–≤–µ–Ω—Å—Ç–≤–æ (!=) ‚ùå

`__ne__` –æ—Ç–≤–µ—á–∞–µ—Ç –∑–∞ –æ–ø–µ—Ä–∞—Ç–æ—Ä —Å—Ä–∞–≤–Ω–µ–Ω–∏—è "–Ω–µ —Ä–∞–≤–Ω–æ". –ü–æ —É–º–æ–ª—á–∞–Ω–∏—é, `__ne__` –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –ø—Ä–æ—Ç–∏–≤–æ–ø–æ–ª–æ–∂–Ω–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ `__eq__`.

–ü—Ä–∏–º–µ—Ä:

```python
        return not self.__eq__(other)

print(circle1 != circle3)  # True
```
